# Reporting Issues

To report a bug, concern, or potential security issue with MxTac, please email
**fankh111@gmail.com** with a clear description of the problem. Do not open a
public GitHub issue for sensitive reports. We will respond as soon as possible
and work with you on a resolution.

## Supported Versions

| Version | Supported |
|---------|-----------|
| v2.x    | ✅ Yes     |
| v1.x    | ❌ No      |
